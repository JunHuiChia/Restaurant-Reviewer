# Restaurant-Reviewer

Restaurant Reviewer is an app that can let you keep track and review your favourite restaurants.

## Features

- Add restaurants
  - Name
  - Price Range
  - Rating
  - Description
- Restaurants show up on the home page
  - Sorted/Categorized by Cuisines
